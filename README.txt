Title: Medical Booking System

Project Introduction
The Medical Booking System is a web application that facilitates the booking of medical appointments, connecting patients with doctors and healthcare facilities. It provides features for searching for available appointments, scheduling appointments, and accessing medical records.

The Medical Appointment Booking System includes the following key functionalities:

- Appointment Management: Efficient searching and scheduling of medical appointments.
- Patient-Doctor Connection: Connects patients with doctors and healthcare facilities.
- Medical Records Access: Provides access to medical records for enhanced patient care.

Deployment Guide (On Local)
Prerequisites
- Java Development Kit (JDK)
- Apache Maven
- MySQL Database

Steps
1. Clone the Repository
git clone https://github.com/Thtdasg97/MedicalBookingSystem-Java_project.git
cd MedicalBookingSystem-Java_project

2. Configure Database

3. Build and run
mvn clean install
mvn spring-boot:run
Access the platform at http://localhost:8080

4. Platform Access:
Open your browser and go to http://localhost:8080
