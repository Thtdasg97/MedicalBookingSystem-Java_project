Title: Medical Booking System

The Medical Booking System is a web application that facilitates the booking of medical appointments, connecting patients with doctors and healthcare facilities. It provides features for searching for available appointments, scheduling appointments, and accessing medical records.

Technologies used

Back-end: Java, Spring Framework, Hibernate, MySQL

Responsibilities

Analyzing project requirements and designing the database using MySQL.
Developing and managing RESTful APIs to facilitate communication and interactions with external systems.
Managing API requests and responses, ensuring accurate data validation and error handling.
Generating Json Web Tokens for secure authentication and authorization.
